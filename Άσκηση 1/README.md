# Σχεδίαση και Χρήση Βάσεων Δεδομένων | Entity-Relationship Diagram Design

Η συγκεκριμένη εργασία αφορά τον σχεδιασμό και την υλοποίηση μιας σχεσιακής βάσης δεδομένων χρησιμοποιώντας το MySQL Workbench αναφορικά με το πρόβλημα της διαχείρισης των προσφυγικών ροών και της εξασφάλισης ποιοτικών συνθηκών διαβίωσης των προσφύγων.

## Περιγραφή 
Το διάγραμμα απεικονίζει τις οντότητες, τα primary και foreign keys, τις σχέσεις μεταξύ οντοτήτων και τους ενδιάμεσους Πίνακες που χρησιμοποιούνται για την υλοποίηση σχέσεων Ν:Μ. Επιπλέον συμπεριλαμβάνει τις γενικεύεσεις (ISA relationship) των οντοτήτων (π.χ. Εθελοντής-Εθελοντές Γιατροί, Εθελοντές Άλλοι).

## Σχέσεις μεταξύ οντοτήτων 
- Οικογένεια - Πρόσφυγας 1:Ν
- Πρόσφυγας - Πρόσφυγας
- Πρόσφυγας - Αίτημα 1:Ν
- Αίτημα - Κυβερνητικοί Φορείς Ν:1
- Κυβερνητικοί Φορείς - Σχολεία 1:Ν
- Κυβερνητικοί Φορείς - Νοσοκομεία 1:Ν
- Νοσοκομεία - Νοσηλεία 1:Ν
- Νοσηλεία - Παραπεμπτικό Ν:1
- Εθελοντές Γιατροί- Παραπεμπτικό 1:Ν
- Εθελοντές Γιατροί - Συνταγογράφηση 1:1 (?)
- Συνταγογράφηση - Φάρμακα 1:1 (?)
- Εθελοντές Γιατροί - Εθελοντής 1:1
- Εθελοντές Γιατροί - ΜΚΟ Υγείας Ν:1
- ΜΚΟ Υγείας - ΜΚΟ Προμηθευτές 1:Ν
  - ΜΚΟ Προμηθευτές - Προμηθευτές Ν:1 (Ενδιάμεσος πίνακας ``ΜΚΟ Προμηθευτές`` για την επίτευξη σχέσης Ν:Μ)
- ΜΚΟ Προμηθευτές Σίτισης - Προμηθευτές Ν:1
  - ΜΚΟ Προμηθευτές Σίτισης- ΜΚΟ Σίτισης/Στέγασης Ν:1 (Ενδιάμεσος πίνακας ``ΜΚΟ Προμηθευτές Σίτισης`` για την επίτευξη σχέσης Ν:Μ)
- Προμηθευτές - Προμήθειες 1:Ν
- Προμήθειες - Φάρμακα (?)
- Προμήθειες - Ένδυση/Τρόφιμα (?)
- Εθελοντής - Εθελοντές Άλλοι 1:1
- Εθελοντές Άλλοι - Λίστα Ειδικοτήτων_has_Εθελοντές 1:Ν
  - Λίστα Ειδικοτήτων_has_Εθελοντές - Λίστα Ειδικοτήτων Ν:1 (Ενδιάμεσος πίνακας ``Λίστα Ειδικοτήτων`` για την επίτευξη σχέσης Ν:Μ)
- Εθελοντές Άλλοι - ΜΚΟ Εθελοντές 1:Ν
  - ΜΚΟ Εθελοντές - ΜΚΟ Σίτισης/Στέγασης Ν:1 (Ενδιάμεσος πίνακας ``ΜΚΟ Εθελοντές`` για την επίτευξη σχέσης Ν:Μ)
- ΜΚΟ Σίτισης/Στέγασης - Συσχέτιση_Προσφύγων_ΜΚΟ 1:Ν
  - Συσχέτιση_Προσφύγων_ΜΚΟ - Πρόσφυγας Ν:1 (Ενδιάμεσος πίνακας ``Συσχέτιση_Προσφύγων_ΜΚΟ`` για την επίτευξη σχέσης Ν:Μ)
- ΜΚΟ Σίτισης/Στέγασης - Κέντρα Φιλοξενίας 1:Ν
- ΜΚΟ Σίτισης/Στέγασης - Ιδιωτικά Σπίτια 1:Ν
-Ιδιωτικά Σπίτια - Οικογένεια 1:1 (?)
